2maiss 🖱️🖱️

Hardware-level dual-mouse touch translation for a specific "washing machine" rhythm game.

LEGAL NOTICE: This repository contains ONLY generic input translation logic. It does NOT distribute proprietary game binaries, decryptors, cabinet keys, or copyrighted Unity/MelonLoader assemblies. It is a clean-room implementation of serial protocols.

Overview

Dedicated hardware touch panels for specific arcade setups can be prohibitively expensive. 2maiss (pronounced Two Mice) bridges the gap by mapping two independent USB mice to a virtual serial port interface, allowing players to execute hardcore charts using standard desktop peripherals.

Technical Architecture

  • Raw Input Interception: Guarantees left and right physical mice do not interfere, isolating movement data at the hardware level.
  • Serial Emulation: Outputs standard touch protocol to com0com virtual pairs (COM33 <-> COM3).
  • Visual Calibration: In-game UI overlay (via F11) to adjust and scale interaction sectors on the fly.
  • Ecosystem Compatibility: Designed to run seamlessly alongside standard environments like AquaMai and MelonLoader.

Deployment Prerequisites

  • AquaMai.toml configuration requires HideSubMonitor = true and BaudRate = 115200.
  • A properly configured com0com loopback.
  • Dedicated Package\Mods\2maiss.dll injection for visual overlays.
